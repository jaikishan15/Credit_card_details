# Credit_card_details

## Create an Android "screen" that would allow a user to enter and submit the following data from an American Express, Discover, MasterCard, or Visa credit card:

##### Credit card number
##### Expiration date (MM/YY format)
##### CVV security code
##### First name
##### Last name
